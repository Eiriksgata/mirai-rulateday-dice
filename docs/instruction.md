# 指令集

> 系统默认 d|D 都是属于100面骰子，所有的指令前缀 `.` 都可以用 `。`中文点(可能有一些指令还没有加入)

## .r
该指令主要用于生成一个骰子的数据。

使用例子:
```
.r1d3+1d5 => 1d3+1d5=1+2=3
.rd=1d100 => 58
.rd+1d3 => D100+1d3=82+3=85
......
```
